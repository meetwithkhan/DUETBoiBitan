CodeCareBD - bKash, Nagad, Rocket, Payoneer Gateway plugin is for WooCommerce. This plugin will help you to integrate bKash, Nagad, Rocket, and Payoneer Payment Gateway. This plugin is very lightweight and easy to use. If you're new to integrating this payment gateway. Then you can follow our below instruction video.

= Demo Video =

[youtube https://www.youtube.com/watch?v=yrK5dhQpX68]


= Please note: =

=> This is a WooCommerce based plugin, so WooCommerce plugin must be activated before active this plugin.
=> You must have a (bKash, Nagad, Rocket, Payoneer) account to get payments.

If you have any idea to improve our plugin you can suggest us or you can also get related to this plugin here: [Support Link](https://codecarebd.net/contact)

Hope you enjoy using this plugin.
